## Key Result

> ADI detects a significant 6.3% decrease in CMR capacity per 10-percentile increase
> in area deprivation (IRR = 0.937, 95% CI: 0.900-0.976, p = 0.002).
> SVI shows no association (IRR = 0.992, p = 0.681).

## Abbreviations

- CMR = Cardiac Magnetic Resonance (Cardiac MRI)
- CCT = Cardiac Computed Tomography (Cardiac CT)
- ACR = American College of Radiology
- SVI = Social Vulnerability Index (CDC)
- ADI = Area Deprivation Index (constructed via PCA)
- IRR = Incidence Rate Ratio
- CI = Confidence Interval
- RUCC = Rural-Urban Continuum Code
- PCA = Principal Component Analysis
